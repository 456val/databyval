Personal Finance Insights Dashboard (ETL + SQL + Power BI)

Project Overview

Managing personal finances can be overwhelming without clear insights. This project addresses the challenge by transforming raw transaction data into an interactive dashboard. The goal is to provide a comprehensive view of income, expenses, and cash flow, enabling informed financial decisions and highlighting areas for optimization.

Methodology

  • Data Cleaning (Excel): Removed duplicates, standardized categories, and handled missing values for accuracy.
  • Data Transformation (Python/Pandas): Categorized transactions, computed balances, and formatted dates for ingestion into SQL.
  • Database Management (SQL): Designed relational schema and aggregated data for efficient querying and reporting.
  • Visualization (Power BI): Built interactive dashboards to analyze inflow vs. outflow, balance trends, and spending patterns.
