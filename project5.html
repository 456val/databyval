Data Transformation

Project Overview

This project focuses on cleaning and transforming a one-day supply transaction dataset that contains multiple product categories and amounts merged within single cells. The goal is to restructure the data into a clear and analyzable format, making it easier to identify spending patterns and transaction summaries. Through this process, the project demonstrates the importance of data cleaning and organization in turning messy raw data into valuable insights.

Objectives

  • To clean and transform merged transaction data into a structured format.
  • To align each product category with its correct transaction amount.
  • To prepare the data for clear analysis and visualization of spending patterns.

Techniques and Tools Used

  • Techniques: Data Cleaning, Data Transformation, Data Visualization, Descriptive Statistics
  • Tools: Python (Pandas, Matplotlib)

✨ This transformation turned disorganized transaction data into a structured format for effective spending analysis.
